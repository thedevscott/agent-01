# AI Agent
A custom simplified code assistant that relies on OpenRouter via the official 
OpenAI Python SDK.

**THIS IS FOR LEARNING PURPOSES ONLY**

# Dependencies
OpenRouter
Python 3.10+
uv

# Setup
## .env file
  OPENROUTER_API_KEY='key'

## OpenRouter
* Create an account on OpenRouter.
* Go to the Keys page and Create a Key. Give it a name like ai-agent.
* Copy and Paste the key into your .env file ```code
  OPENROUTER_API_KEY='api_key' ``` replace "api_key" with your copied key.
